# DDSimpleRSS

An iPhone app project that build a simple RSS reader.  The main purpose is easy to integrate with the project you are working on.

## Features:

* Loading view
* Pull to refresh
* Parse RSS
* Load remote image
* Detail screen view

## External libraries:

* DDTableViewPullRefresh
* HJCacheClasses
* LoadingView
* MWFeedParser

## Installation

### Required frameworks

* QuartzCore
* UIKit
* Foundation
* CoreGraphics

### Disabling ARC

Add to compiler flag: `-fno-objc-arc`
